# ProEdit

ProEdit is a modular .NET office suite and document engine built around Avalonia, ReactiveUI, and Skia. It provides document models, layout and rendering pipelines, editing services, printing and PDF backends, collaboration components, and Avalonia UI shells.

## Build

```bash
dotnet build ProEdit.slnx
```

## Run

```bash
dotnet run --project src/ProEdit.Word.App/ProEdit.Word.App.csproj
```

## Docs
