# VibeOffice

VibeOffice is a modular .NET office suite and document engine built around Avalonia, ReactiveUI, and Skia. It provides document models, layout and rendering pipelines, editing services, printing and PDF backends, collaboration components, and Avalonia UI shells.

## Build

```bash
dotnet build VibeOffice.slnx
```

## Run

```bash
dotnet run --project src/Vibe.Word.App/Vibe.Word.App.csproj
```

## Docs
