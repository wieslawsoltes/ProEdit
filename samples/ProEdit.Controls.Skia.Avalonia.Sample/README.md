# ProEdit.Controls.Skia.Avalonia.Sample

Avalonia desktop sample for `ProEdit.Controls.Skia.Avalonia`.

```bash
dotnet run --project samples/ProEdit.Controls.Skia.Avalonia.Sample/ProEdit.Controls.Skia.Avalonia.Sample.csproj
```

The sample displays `ProEditDocumentViewer` and `ProEditDocumentEditor` side by side and binds toolbar commands to the shared control API: zoom modes, view modes, pagination, page flow, pan, gridlines, invisibles, and read-only editing.
