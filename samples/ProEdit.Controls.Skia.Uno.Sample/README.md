# ProEdit.Controls.Skia.Uno.Sample

Uno Platform sample for `ProEdit.Controls.Skia.Uno`.

```bash
dotnet run --project samples/ProEdit.Controls.Skia.Uno.Sample/ProEdit.Controls.Skia.Uno.Sample.csproj
```

The sample displays `ProEditDocumentViewer` and `ProEditDocumentEditor` side by side and binds the same shared sample ViewModel used by the Avalonia and MAUI heads.
