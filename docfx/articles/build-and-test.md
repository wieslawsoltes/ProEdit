# Build and Test

## Build

```bash
dotnet build VibeOffice.slnx
```

## Test

```bash
dotnet test VibeOffice.slnx
```

## Benchmarks

```bash
dotnet run --project benchmarks/Vibe.Office.Layout.Benchmarks/Vibe.Office.Layout.Benchmarks.csproj
```

## Docs

```bash
./build-docs.sh
```

```bash
./serve-docs.sh
```

On Windows:

```powershell
./build-docs.ps1
```

```powershell
./serve-docs.ps1
```
