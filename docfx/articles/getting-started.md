# Getting Started

## Prerequisites
- .NET 10 SDK

## Build

```bash
dotnet build VibeOffice.slnx
```

## Run the Desktop App

```bash
dotnet run --project src/Vibe.Word.App/Vibe.Word.App.csproj
```

## Run Tests

```bash
dotnet test VibeOffice.slnx
```

## Build Docs

```bash
./build-docs.sh
```

On Windows:

```powershell
./build-docs.ps1
```
