# Getting Started

## Prerequisites
- .NET 10 SDK

## Build

```bash
dotnet build ProEdit.slnx
```

## Run the Desktop App

```bash
dotnet run --project src/ProEdit.Word.App/ProEdit.Word.App.csproj
```

## Run Tests

```bash
dotnet test ProEdit.slnx
```

## Build Docs

```bash
./build-docs.sh
```

On Windows:

```powershell
./build-docs.ps1
```
